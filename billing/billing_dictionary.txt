MAPEAMENTO DETALHADO DE CAMPOS
====================================================================================================

Formato: origem : billings.<campo>  (aplicar em gmfinancial.billings HML e financial.billings PRD)

  [GERADO AUTOMATICAMENTE] : billings.id

  [PREENCHIDO PELA LOGICA DE MIGRACAO - IdCliente (view) -> hash do IdCliente canônico deduplicado (CUST-001/BILL-003) -> gmcore.customers.legacy_ids -> gmcore.customers.id] : billings.customer_id

  [PREENCHIDO PELA LOGICA DE MIGRACAO - prioridade: View (nome cliente orcamento) > XLSX.nome_cliente > gmcore.customers.legal_name/trade_name] : billings.name

  [PREENCHIDO PELA LOGICA DE MIGRACAO - sempre NULL (informacao inexistente na origem)] : billings.observations

  [PREENCHIDO PELA LOGICA DE MIGRACAO - sempre NULL na carga inicial] : billings.deleted_at

  [PREENCHIDO PELA LOGICA DE MIGRACAO - XLSX.orcamento_ativo (bool -> is_active); celula vazia/null: default true ou conforme regra] : billings.is_active

  [PREENCHIDO PELA LOGICA DE MIGRACAO - datetime migracao ou origem se disponivel] : billings.created_at
  [PREENCHIDO PELA LOGICA DE MIGRACAO - datetime migracao ou origem se disponivel] : billings.updated_at

  [PREENCHIDO PELA LOGICA DE MIGRACAO - XLSX.tipo_faturamento_descricao ("4 Week" -> four_weeks_in_month; "5 Week" -> five_weeks_in_month); vazio ou NULL na coluna: four_weeks_in_month] : billings.calculation_type

  [PREENCHIDO PELA LOGICA DE MIGRACAO - XLSX.dia_faturamento (origem obrigatoria); celula vazia/null: default conforme DDL ou regra] : billings.contract_parameters_billing_day

  [PREENCHIDO PELA LOGICA DE MIGRACAO - XLSX.dia_vencimento (origem obrigatoria); celula vazia/null: default conforme DDL ou regra] : billings.contract_parameters_due_day

  [PREENCHIDO PELA LOGICA DE MIGRACAO - FASE 1: sempre current; depois mapear vigente/antecipado/retroativo] : billings.contract_parameters_billing_type

  [PREENCHIDO PELA LOGICA DE MIGRACAO - sempre monthly (BILL-001 fechado)] : billings.contract_parameters_thirteenth_salary_type
